Flat XML from file

In this example a file URL returning a XML in a flat structure is passed at the initialization. The XML is transformed and used to populate the tree. Read more about the expected XML format.

This format is useful for displaying a id -> parent_id structure from a database, because the client script does the ordering & nesting.
A query like SELECT * FROM table ORDER BY parent_id, position is sufficient.

javascript$(function() { tree1 = new tree_component(); tree1.init($("#demo"), { data : { type : "xml_flat", url : "flat.xml" } }); }); html<div id="demo"></div> xml<?xml version="1.0" encoding="UTF-8"?> <root> <item id="1" parent_id="0" state="open"><content><name>Root node</name></content></item> <item id="2" parent_id="1"><content><name icon="../images/globe.png">Child 1</name></content></item> <item id="3" parent_id="1" title="My name is child 2"><content><name>Child 2</name></content></item> <item id="4" parent_id="2"><content><name>A sub child</name></content></item> <item id="5" parent_id="2"><content><name>Sub-children ...</name></content></item> </root>
