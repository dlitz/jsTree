A of version 0.9.5 the native context menu is enabled by default. Here is how you disable it. For adding more items to the context menu - check the documentation (the context menu part in the config section).



javascript$(function() { tree1 = new tree_component(); tree1.init($("#demo"), { data : { type : "xml_flat", url : "flat.xml" }, ui : { animation : 500, rtl : false, dots : true, context : false } }); });
