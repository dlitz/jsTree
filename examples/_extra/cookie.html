Cookie support

Optionally you could enable the cookie extension. It uses the jquery cookie plugin.

Just pass na object for settings.cookies - the object must contain a property called "prefix" - this is used for the cookie name (it's a good way to distinquish between different trees in multitree, the second property of the object is the "opts" object - it is the standard cookie plugin options object (expires, path, domain, secure).

If the cookies object is set the tree will create two cookies - "<prefix>_selected" and "<prefix>_open". The first one contains the ID of the selected node, the second contains the comma-separated IDs of the open nodes.

*  NOTE: try clicking around and check the values of the cookies.
