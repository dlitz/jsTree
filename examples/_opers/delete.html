Deleting nodes

This is the xml_flat example with the addition of rules for deleting nodes.

The settings.rules.deletable object can either be set to "all", "none" or an array of string values. Each of the items of the array is compared against the settings.rules.type_attr attribute (which defaults to "rel").

In this example the user can delete only nodes of type "folder".

* You could also check the metadata examples for inline declaration of createable.

You can pass an argument to the remove function to delete a specific node, otherwise the selected node is deleted.
